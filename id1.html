    Share × Credits ×

correr

[ koˈher ] Full verb table transitive verb 1.  to run 2.  ( viajar por ) to travel across 3.  ( cortina ) to draw 4.  ( expulsar ) to drive out Full verb table intransitive verb 1.  to run 2.  ( em carro ) to drive fast , speed 3.  ( líquido ) to flow , run 4.  ( o tempo ) to elapse 5.  ( boato ) to go round 6.  ( atuar com rapidez ) to rush está tudo correndo bem everything is going well as despesas correrão por minha conta I will handle the expenses
